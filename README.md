# ClipboardTool 剪贴工具⌨

## 简短介绍

ClipboardTool 是一个基于 C# 5 和 WPF（.NET Framework 4.7.2）的 Windows 桌面剪贴板增强工具,具有多种便捷功能

核心能力：模拟复制/剪切/粘贴快捷键、截图与 OCR 文字识别、剪贴板历史记录管理、常用信息收藏、以及集成多种系统工具快捷启动的工具箱

---

## 创作动机

经常需要复制剪贴粘贴截图需要频繁切换鼠标键盘,苦于其他输入法复制历史条数上限,于是开发了这款软件

>依旧懒是第一生产力

## 如何使用

### 编译方式

**使用 Build.bat**：双击 `Code\Build.bat`，脚本会生成 `ClipboardTool.exe`,运行即可

### 运行环境要求

- Windows 操作系统(Windows 10/11)
>更低版本需要安装.NET Framework 4.7.2
- OCR 功能依赖 `tessdata\` 目录下的训练数据文件（已内置简体中文 `chi_sim` 和英文 `eng`）
- OCR 功能依赖 `Tesseract.dll`、`tesseract41.dll`、`leptonica-1.80.0.dll`（已随程序附带，含 x64 原生库）

### 基本操作

| 操作 | 说明 |
|------|------|
| **拖动窗口** | 在窗口任意空白处按住鼠标左键拖动 |
| **切换页面** | 点击标题栏左右的 `‹` / `›` 箭头按钮，在四个页面间循环切换 |
| **打开设置** | 在主页面点击 `⋯` 按钮，展开设置面板 |
| **关闭程序** | 点击 `×` 按钮 |
>好大一坨废话😅 ↑

## 功能

### 一、主页面 — 快捷操作按钮🤓👆

提供四个核心操作按钮，点击后程序会自动隐藏自身、激活上一个前台窗口，并模拟键盘快捷键执行操作：

| 按钮 | 功能 | 模拟按键 |
|------|------|----------|
| 复制 | 对当前选中内容执行复制 | `Ctrl + C` |
| 剪切 | 对当前选中内容执行剪切 | `Ctrl + X` |
| 粘贴 | 将剪贴板内容粘贴到当前光标位置 | `Ctrl + V` |
| 截图 | 调用 Windows 截图工具根据设置提取图片或进行 OCR 文字识别 | `Win + Shift + S` |

> 键盘不再是关键（扔）

### 二、剪贴板页面 — 历史记录管理📋

- **自动记录**：每 500 毫秒轮询一次剪贴板，自动捕获新的文本内容并置顶记录（去重）
- **点击粘贴**：点击任意历史记录卡片，自动将内容写入剪贴板并模拟 `Ctrl + V` 粘贴到上一个前台窗口
- **搜索**：点击底部「搜索」按钮切换搜索模式，实时过滤历史记录
- **跳转**：在底部输入框中输入序号并回车，快速定位到对应记录
- **删除**：点击卡片右侧的 `−` 按钮，二次点击确认删除
- **容量限制**：默认最多保存 255 条，可在设置中自定义
- **持久化**：退出时自动将历史记录以 Base64 编码保存到 `ClipboardToolData\ClipboardTool.history`，下次启动自动恢复
>太棒了!!!
### 三、常用信息页面 — 固定文本收藏✨

- **添加**：点击底部 `＋` 按钮，在弹出的对话框中输入常用文本（支持多行）
- **点击粘贴**：点击卡片即可将内容复制到剪贴板并粘贴
- **排序**：每张卡片右侧提供 `↑` 上移和 `↓` 下移按钮，可调整顺序
- **删除**：点击 `−` 按钮删除条目
- **跳转**：支持按序号快速跳转
- **持久化**：数据保存在 `ClipboardToolData\ClipboardTool.common`

### 四、工具箱页面 — 快捷工具启动🛠

内置多种 Windows 系统工具的快捷启动入口，按使用频率排序：

| 工具 | 说明 |
|------|------|
| 管理员 Shell | 以管理员权限启动 PowerShell |
| CMD | 启动命令提示符 |
| 桌面键盘 | 启动屏幕键盘（osk.exe） |
| 字符库 | 启动字符映射表（charmap.exe） |
| 语音输入 | 模拟 `Win + H` 唤起语音输入 |
| 表情库 | 模拟 `Win + .` 唤起表情面板 |
| 计算器 | 启动计算器（calc） |
| 记事本 | 启动记事本（notepad） |
| 任务管理器 | 启动任务管理器（taskmgr） |
| 控制面板 | 启动控制面板（control） |
| 资源管理器 | 启动资源管理器（explorer） |
| 倒计时 | 自定义分钟数倒计时，到时弹窗提醒，运行中再次点击可取消 |
| 番茄钟 | 25 分钟工作 / 5 分钟休息循环计时，运行中再次点击可停止 |

- 使用次数会被记录并持久化到 `ClipboardToolData\ClipboardTool.toolbox`，工具按使用频率从高到低排序

### 五、截图与 OCR 文字识别

截图功能调用 Windows 系统截图工具后，会自动轮询剪贴板，获取截图后提供以下可选处理：

- **截图提取文字（OCR）**：开启后，截图完成后自动识别图片中的文字，识别结果自动写入剪贴板历史并弹出提示。识别过程在后台线程执行，不阻塞界面
>文字识别支持简体中文和英文
- **截图提取图片**：开启后，在剪贴板历史中插入一条 `[截图]` 记录
- **截图存入桌面**：开启后，将截图以 PNG 格式保存到桌面，文件名格式为 `截图_yyyyMMdd_HHmmss.png`

### 六、设置面板

在主页面点击 `⋯` 按钮展开，提供以下个性化设置：

#### 颜色定制
- **背景颜色**：18 种可选背景色（含深色与浅色变体）
- **按钮颜色**：10 种按钮主题色方案（粉/红/橙/黄/绿/青/蓝/紫/黑/白）
- **渐变效果**：开启后按钮使用线性渐变填充
- **文字颜色**：12 种文字颜色可选
>"怎么没有五颜六色的黑"

#### 行为开关
| 开关 | 说明 |
|------|------|
| 置顶 | 窗口始终不会被其他窗口遮盖 |
| 边缘吸附 | 拖动窗口靠近屏幕边缘时自动吸附 |
| 截图提取文字 | 截图后自动进行 OCR |
| 截图提取图片 | 截图后保存到历史记录 |
| 截图存入桌面 | 截图后保存 PNG 到桌面 |
| 半透明 | 开启窗口半透明效果，可调节透明度（10%–100%） |

#### 其他
- **最大记录数**：自定义剪贴板历史记录上限
- 所有设置实时保存到 `ClipboardToolData\ClipboardTool.settings`

### 七、界面交互细节

- **窗口无边框**：采用 `WindowStyle.None` + `AllowsTransparency`，自绘标题栏与关闭按钮
- **页面切换动画**：切换页面时窗口高度平滑过渡，内容区域左右滑动 + 淡入淡出
- **卡片入场动画**：列表卡片在滚动进入可视区域时逐个淡入
- **按钮交互动画**：悬停放大、按下缩小、阴影变化等微交互
- **粘贴反馈**：点击历史记录粘贴时，卡片产生缩放动画反馈
- **启动动画**：程序启动时窗口淡入显示
